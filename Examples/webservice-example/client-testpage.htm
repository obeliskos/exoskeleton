Webservice Test Client Page

This is an example of a webpage (which is not running under exoskeleton) to call web services within an exoskeleton application. For this particular example we will serve this page up from the self-hosting exoskeleton web server functionality but it could in theory be served up from another web server.

This page will GET GetContactList.svc immediately on first load to get list of contacts to populate list with. When you select a contact from the list it will call GetContact.svc, passing the contact 'name' as a POST parameter and receive a json object as a return value.
