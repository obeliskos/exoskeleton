About Exoskeleton Interactive Console

This is a simple interactive script runner which lets you experiment with exoskeleton. Technically, this is not a sandbox since exoskeleton's whole purpose is to break out of the sandbox, so use responsibly.

This utility page is to support the Exoskeleton project.

Some example logging commands are :
xo.logText(message);
xo.logObject(obj);

This utilizes CodeMirror for the editors and they have code folding, and autocomplete functionality (available using CTRL-Space). AutoComplete can allow to explore exoskeleton classes and functions.
